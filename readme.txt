== Description ==
Slider Path for Elementor to enrich widgets features of elementor.Slider Path will give you unlimited options of slider choice. you may edit control of the full slider with the default style. All category has Different Slider.

# Features Includes

-	Number or Ready Sliders
-	Automatic Settings and Style
-	Custom of Elementor and Slides
-	Responsive and Device Friendly Design
-	Get Support Instant
-	All Category of Slider you will get
-	NO Coding Requers
-	Easy Demo Import
-	Cron Job Demo Import tLibrary
-	Modern Design

# How to use it

-	Go to Library 
-	Check the Slider 
-	Preview the Slider
-	Import and Ready to Use
